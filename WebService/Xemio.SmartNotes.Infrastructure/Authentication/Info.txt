Here's some pseudo code to authenticate a request (you need the username, the password and the content that you want to authenticate).

byte[] authenticationBytes = UTF8.GetBytes(Username + Password);
byte[] authenticationHash = SHA256(authenticationBytes);

byte[] contentBytes = UTF8.GetBytes(Content);
byte[] contentHash = HMACSHA256(contentBytes, authenticationHash);
string contentHashString = Base64(contentHash);

Pass this "contentHashString" in the Authorization request header in the format "Xemio Username:contentHashString"

Some example values, so you can check your code if it works correctly:

var Username = "TestUsername";
var Password = "123456";
var Content = "TestContent";
